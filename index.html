Тестирование на проникновение

Внешний пентест

Комплексная оценка защищенности внешнего периметра вашей инфраструктуры. Тестирование включает анализ открытых портов, сервисов, веб-приложений и поиск путей проникновения извне.

  • Сканирование портов и сервисов
  • Анализ веб-приложений
  • Тестирование SSL/TLS
  • Проверка DNS-записей

Внутренний пентест

Тестирование безопасности внутренней сети и выявление путей эскалации привилегий. Моделирование атак изнутри корпоративной сети для оценки внутренних угроз.

  • Латеральное движение по сети
  • Эскалация привилегий
  • Анализ Active Directory
  • Тестирование сегментации

Приложения & Сервисы

Комплексное тестирование безопасности приложений всех типов: веб, мобильные, десктоп, API, IoT устройства. Анализ согласно современным методологиям и стандартам безопасности.

  • Веб и мобильные приложения
  • API и микросервисы
  • Десктоп приложения
  • IoT устройства

Red Team

Комплексное моделирование реальных атак на вашу инфраструктуру. Многовекторные атаки с использованием передовых техник для проверки готовности к APT-угрозам.

  • APT-сценарии атак
  • Социальная инженерия
  • Физическое проникновение
  • Обход средств защиты
